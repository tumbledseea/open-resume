# ATS Checker

The ATS check should prefer simple, parseable resumes.

## Checkpoints

- Contact information appears near the top.
- Standard section names are used.
- Bullets are plain text.
- No tables, images, text boxes, or multi-column layout in the Markdown source.
- Keywords from the JD are covered when a JD exists.
- One-page Chinese resume target is roughly 900-1400 Chinese characters unless the user requests otherwise.
