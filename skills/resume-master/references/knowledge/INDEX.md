# 简历知识库 Knowledge Base

本目录是 resume-master skill 的可检索知识资产，原文来自 WonderCV 开源仓库
（resume-guide、resume-skills-and-tools），按主题归类。agent 在分析 JD、制定策略、
写作 bullet、做 ATS 优化或准备面试材料时，应先检索这里的对应文档作为参考。

> 用途定位：这些是**写作方法论与示例**，不是事实来源。简历内容仍只能取自候选人
> 的 `profile.json` / `fact_index.json`。知识库只影响表达方式、结构和关键词策略。
