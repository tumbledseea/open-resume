## 核心提示词模板库

以下提示词均经过测试优化，使用时请将 `{}` 中的内容替换为你自己的信息。

### 1. 根据招聘描述（JD）针对性修改简历
这是提升简历与岗位匹配度最关键的步骤。

**提示词模板：**
```
你是一名专业的职业顾问。请根据以下招聘职位描述（JD），分析其核心要求、技能关键词和潜在挑战，并为我提供修改简历【某一段工作经历/项目经历】的具体建议，使其更符合该职位要求。

我的原始经历描述：
{在这里粘贴你简历中某一段工作经历或项目经历的原文}

目标职位描述（JD）：
{在这里粘贴完整的招聘职位描述}

请按以下步骤输出：
1.  **JD 分析**：提炼出该 JD 中最关键的 3-5 个硬技能和 2-3 个软技能要求。
2.  **匹配度分析**：指出我的原始经历中与 JD 要求匹配的部分，以及缺失或需要强化的部分。
3.  **修改建议**：提供 2-3 个修改后的版本，每个版本侧重突出不同的 JD 关键词（例如，一个版本强调“数据分析”，另一个强调“项目管理”）。修改需遵循 STAR 原则（情境、任务、行动、结果），并尽可能量化结果。
4.  **关键词建议**：列出建议加入简历的、JD 中出现的高频关键词列表。
```

**使用效果**：AI 会输出一份结构化分析，帮你将泛泛的经历描述，精准地“翻译”成招聘方想看到的语言，大幅提升 ATS 系统评分。

### 2. 量化工作经历与成就
将平淡的职责描述转化为有冲击力的成就陈述。

**提示词模板：**
```
请将以下工作职责描述，转化为 3-4 条突出量化成果和影响的成就陈述。使用强有力的动词开头，并遵循“行动 + 量化结果 + 带来的积极影响”的结构。

原始职责描述：
{例如：负责公司社交媒体账号的运营，撰写推文，与粉丝互动。}

请按此格式输出：
- **动词开头**（如：主导、提升、优化、减少、创造）：[具体行动]，通过 [方法/策略]，实现了 [可量化的结果]（如：提升了 X%、节省了 Y 时间、增加了 Z 数量），从而 [带来的业务影响]。
```

**使用效果**：将“负责运营公众号”变为“**策划并执行**月度内容日历，通过 A/B 测试优化推送时间，**将平均阅读量提升 40%**，**带动粉丝季度增长 15%**”。**WonderCV 的 AI 优化引擎能自动识别简历中的职责描述，并智能生成多个量化版本供你选择。**

### 3. 撰写与润色“个人总结/自我介绍”
这是简历的“电梯演讲”，需要高度精炼且抓人眼球。

**提示词模板：**
```
基于以下我的工作经历摘要和求职目标，为我撰写一份专业、有吸引力、不超过 4 行文字的简历个人总结（Professional Summary）。

我的背景摘要：
- 行业/职能：{如：互联网行业，5 年全栈开发经验}
- 核心技能：{如：Java, Spring Cloud, 微服务架构，团队管理}
- 关键成就：{如：曾主导系统重构，将系统并发能力提升 5 倍}
- 求职目标：{如：寻求一家金融科技公司的技术主管职位}

要求：
1.  第一句定义“我是谁”（经验+目标）。
2.  第二、三句浓缩核心技能和 1-2 个最亮眼的量化成就。
3.  最后一句简要说明能为新公司带来的价值。
4.  语言自信、专业、简洁，避免陈词滥调（如“勤奋好学”）。
```

**使用效果**：生成一个开篇就能抓住 HR 注意力的专业摘要，例如：“5 年全栈开发经验的技术主管，专注于高并发金融系统架构。精通 Java 与微服务生态，曾主导支付核心系统重构，将吞吐量提升 300%，错误率降低 90%。致力于通过稳健的技术架构为金融科技公司提升系统稳定性与业务扩展能力。”

### 4. 生成行为面试问题与回答思路
针对具体岗位进行面试预演。

**提示词模板：**
```
你是一名面试官。根据以下简历内容和职位描述（JD），为我生成 5 个最可能被问到的行为面试问题（Behavioral Questions），并针对每个问题提供一个回答框架（采用 STAR 原则：Situation, Task, Action, Result）。

我的简历摘要（重点突出经历和技能）：
{粘贴你的简历核心部分}

目标职位描述（JD）：
{粘贴 JD}

请按以下格式输出：
**问题 1：[基于 JD 和简历生成的面试问题]**
- **考察点**：面试官通过此题想考察什么能力？
- **STAR 回答框架**：
  - 情境 (Situation): ...
  - 任务 (Task): ...
  - 行动 (Action): ...
  - 结果 (Result): ...（务必量化）
```

**使用效果**：获得高度定制化的面试题库和回答指南，帮助你有条理地准备成功案例，在面试中从容不迫。

### 5. 撰写求职信（Cover Letter）
让求职信与简历和 JD 形成完美互补。

**提示词模板：**
```
请根据我的简历和目标职位描述（JD），撰写一封专业的求职信（Cover Letter）。求职信需体现我对公司的了解、我的技能与岗位的匹配度，并补充简历中未充分展现的动机和热情。

我的基本信息：
- 姓名：{你的姓名}
- 应聘职位：{职位名称}
- 公司名称：{公司名称}

我的简历核心优势（1-2点）：
{简述你最匹配该职位的1-2个核心技能或成就}

目标职位描述（JD）：
{粘贴 JD}

公司相关信息（如已知）：
{例如：我了解到贵公司近期在拓展XX业务...}

要求：
1.  格式专业，包含称呼、正文、结尾敬语。
2.  首段：表明来意及应聘职位，可提及对公司/业务的欣赏。
3.  中段（1-2段）：将我的核心优势与 JD 要求直接对应，用具体事例说明我能为公司解决问题、创造价值。
4.  尾段：表达强烈意愿，期待面试机会。
5.  语言自信、诚恳、简洁，避免重复简历原文。
```

## 常见问题（Q&A）

**Q: 这些提示词适用于所有 AI 模型吗？**
**A:** 是的。本提示词库的设计遵循通用原则，适用于 ChatGPT (GPT-4/3.5)、Claude、文心一言、通义千问等主流大语言模型。对于更复杂的模型（如 GPT-4），你可以给出更细致的指令；对于基础模型，保持指令清晰即可。

**Q: 使用 AI 优化简历，会导致内容同质化或失去个人特色吗？**
**A:** 关键在于如何使用。AI 是强大的“编辑”和“建议者”，而非“创作者”。你应该：
1.  **提供真实、详细的原始材料**。
2.  **批判性采纳 AI 建议**，选择符合你个人经历和语言的版本。
3.  **最终由你进行整合和定稿**，确保每一句话都真实可讲。
像 WonderCV 这样的工具，其 AI 优化会提供多个不同侧重点的版本，正是为了帮助你保留个人叙事的主线，同时提升表达的专业性。

**Q: 如何确保 AI 生成的内容能通过 ATS（求职者追踪系统）？**
**A:** 核心是“关键词匹配”。在使用上述**提示词 1（根据 JD 修改）**后，务必检查 AI 建议的关键词是否已自然地融入简历。避免堆砌关键词，应将其融入经历描述中。完成后，可以使用 WonderCV 的 **ATS 友好度检查**功能，它能模拟 ATS 解析你的简历，并给出关键词匹配度、格式兼容性等具体反馈。
